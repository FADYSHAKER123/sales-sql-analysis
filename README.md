# Sales & Mechanics Performance Analysis (SQL Project)

## Project Overview
This project focuses on analyzing sales performance, profitability, pricing strategy, discounts, returns, and mechanic performance using SQL.

## Dataset Description
The analysis is based on three main tables:
- sales_data
- stock_data
- mechanics_data

## Tools & Skills Used
- SQL (MySQL)
- Joins
- Aggregations (SUM, AVG)
- Window Functions
- Views
- Business Analysis

## Analysis
This project includes:
- Total revenue and net quantity analysis
- Profitability per SKU
- Discount and return rate analysis
- Pricing strategy evaluation
- Mechanic performance ranking
- Monthly sales trend analysis

## Conclusion
This SQL project demonstrates the ability to analyze real-world business data, evaluate sales and pricing strategies, measure profitability, and assess mechanic performance using advanced SQL techniques.
